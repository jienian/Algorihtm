### 1.1.1滚动数组
``` java
// 1、1、2、3、5、8、13、21、34、……
private int fibonacci(int n) {// n>=0
    if (n == 0 || n == 1)
        return 1;
    int[] num = new int[n + 1];
    num[0] = 1;
    num[1] = 1;
    for (int i = 2; i <= n; i++)
        num[i] = num[i - 1] + num[i - 2];
    return num[n];
}

```
